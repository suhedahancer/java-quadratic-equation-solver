# java-quadratic-equation-solver
Java GUI Quadratic Equation Solver Project
Bu proje Java ile yapılmış bir ikinci derece denklem çözücüdür.

Program ax² + bx + c = 0 şeklindeki denklemleri çözer.

Kullanım:
1. a, b ve c değerlerini girin
2. Solve butonuna basın
3. Sonuç ekranda gösterilir

Özellikler:
- Kullanıcı dostu GUI
- İkinci derece denklem çözümü
- Java Swing kullanılmıştır

Teknolojiler:
- Java
- Swing GUI
